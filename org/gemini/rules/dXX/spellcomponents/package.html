The org.gemini.rules.dXX.spellcomponents package provides the SpellComponent interface for defining the components necessary to cast a spell. This is done via the SpellComponent interface to do so. An abstract implementation is provided for convenience. Additionally, the most common components (divine focus, focus, material, somatic, verbal) are provided. One additional SpellComponent implementor is provided, XPCostComponent. Given that XP Cost applies to other things (crafting, etc.), it is provided outside the spellcomponents package.
