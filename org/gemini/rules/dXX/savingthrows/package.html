The org.gemini.rules.dXX.savingthrows package provides the SavingThrowEffect interface for defining the effect of a saving throw and pairing it with the type of save (Fortitude, Reflex, Will). This is done via the SavingThrowEffect interface to do so. An abstract implementation is provided for convenience. Additionally, the most common effects (Disbelief, Half, Negates, Partial) are provided.
