The org.gemini.rules.dXX.psionicdisplays package provides the PsionicDisplay interface for defining the "display" of psionic powers in dXX. Additionally, an abstract implementation is provided for convenience, and each of the five standard displays (Auditory, Material, Mental, Olfactory, Visual) are implemented.
